# Карты подскажут

Проект — одностраничное адаптивное веб-приложение, посвящённое колоде
`Oblique Strategies`. Цель: показать содержимое статей, цитат, и элементы
интерфейса (тема, меню, карточка, список).

## Структура

- `index.html` — основная разметка страницы
- `styles/style.css` — каскадные стили (темы, компоненты, адаптивность)
- `scripts/theme-changer.js` — переключатель тёмной/светлой темы (закомментирован)
- `fonts/`, `images/` — ресурсы проекта

## Особенности

- Используются CSS-переменные для темизации
- Адаптивное изображение через `<picture>` с `srcset` и `loading="lazy"`
- Стили для текста, цитат, списков, карточек, первой буквы
- Меню с переключателем темы и индикацией активного состояния
- Простая структура HTML6 с семантическими элементами (figure, nav, header)

## Коммиты

Проект ведётся в Git с понятными сообщениями, использованы Conventional Commits.

## Развертывание

Открыть `index.html` в браузере. Нет клиентской сборки — всё "vanilla HTML/CSS"
